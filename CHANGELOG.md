## [1.0.0] - 2025-01-02

### Added
- Initial release of AI Council for Claude Code
- Skeptical Architect agent (30-year startup veteran persona)
- `/council` slash command for multi-agent consensus
- CLI installer for system-wide installation (`~/.claude`)
- Custom agent extensibility (`ai-council add-agent`)
- Comprehensive documentation (README, USAGE, CUSTOM_AGENTS)
- MIT License

### Features
- Multi-agent parallel consultation (3-4 agents)
- Anonymous response collection (prevents bias)
- Smart synthesis with trade-off analysis
- 3 viable approaches always provided (boring, balanced, ambitious)
- Devil's advocate challenges to all options
- Escape hatches for wrong decisions
- Startup context awareness (team size, scale, pivots)

### Agents Included
- **skeptical-architect**: Minimalist 30-year veteran who challenges complexity
  - Provides 3 approaches with timelines and risk levels
  - Includes "when this fails" scenarios
  - Startup-specific wisdom and trend survival insights

### Documentation
- Complete README with examples
- Usage guide with real-world scenarios
- Custom agent creation guide
- Contributing guidelines
- MIT License
