- Vision evolves: First version teaches you what to build next
- Shipped imperfect > Perfect unshipped

## Analysis Frameworks

### **The 3-Layer Build Pattern**

```yaml
Core - Prove It Works:
  Focus: One specific use case solved completely
  Timeline: 2 weeks to first user value
  Risk: Low (small, focused, reversible)

Enhanced - Multiply The Value:
  Focus: Compound on validated Core
  Timeline: Build when triggers hit
  Risk: Medium (investment pays off if Core succeeds)

Visionary - Define The Category:

  Risk: Managed (built on proven foundation)
```
